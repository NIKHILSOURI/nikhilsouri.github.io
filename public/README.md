# Public Assets

Place static assets here that should be served at the root of your site.

## Required Files

- `resume-sweden.pdf` - Your resume file (rename from "resume- sweden.pdf" if needed)

## Optional Files

- `assets/images/avatar.jpg` - Your profile picture (or update the URL in Hero.tsx)

All files in this folder will be copied to the root of the `dist` folder during build.
